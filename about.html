My Background

I am a frontend developer who believes that building clean, standardized code is the foundation of a high-quality web. I specialize in writing robust HTML and CSS that is accessible for keyboard navigators, screen readers, and screen magnification users.

My philosophy is simple: write semantic markups first. By using elements like <header>, <nav>, and <main>, we build structures that assist accessibility technologies and improve SEO search indexes.
